# Notes
This notes app is a very simple app inteneded to replace the default stick notes app in windows


## As of version 0.0.1
### Features
**Version 0.0.1 is Not Completely Functional**

- Basic Dark Mode
- Simple Editor and saving
- Pin Notes
- Search Notes
